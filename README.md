# 📝 Task Tracker (C# Console App)

A simple task tracker application built using C# as a beginner project.

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Remove tasks

## 🛠️ Technologies Used
- C#
- .NET Console Application

## 📌 How It Works
The program displays a menu where the user can type commands:

- `add` → to add a new task  
- `view` → to display all tasks  
- `mark` → to mark a task as completed  
- `remove` → to delete a task  
- `exit` → to close the program  

## 💡 Concepts Used
- Arrays
- Loops (while, for)
- Conditional statements (switch, if)
- Functions (methods)

## ⚠️ Notes
- Tasks are stored in a fixed-size array (max 100 tasks)
- Completed tasks are displayed but not stored with a status
- The program is case-insensitive for commands
